LIB2024- QA

  1. Qus: What are the three components of Web?

    Ans: The Three components of the web are: HTML, CSS, and JavaScript.

  2. Qus: Define version control and give one example.

    Ans: Version control is a system that keep a record of all changes to the file, so all versions of the files will be remebered. An example of a version control software is GitHub.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag used to create the title of a web page is <title> and </title>. For example: <title>This is the title of the web page</title>

  4. Qus: What tag is used to create a paragragh in html? Give an example.

    Ans: The tag used when creating a paragraph is <p> and </p>. For example: <p>This is a paragraph</p>
